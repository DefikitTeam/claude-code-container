Step 1: Fork the Repository

Create your own copy of the Claude Code Container repository to deploy to your Cloudflare account.

Fork Repository on GitHub

Step 2: Configure Secrets

Add your Cloudflare and Anthropic credentials to enable deployment.

Required Secrets:

  • CLOUDFLARE_API_TOKEN
  • CLOUDFLARE_ACCOUNT_ID
  • ANTHROPIC_API_KEY
  • ENCRYPTION_KEY
Go to Settings → Secrets and variables → Actions in your forked repository

Step 3: Automatic Deployment

Once secrets are configured, push any change to trigger automatic deployment via GitHub Actions.

95%+ Success Rate

GitHub Actions provides reliable deployment for complex Worker projects

Why not a simple deploy button?

This project uses advanced Cloudflare features (Containers + Durable Objects) that require complex build processes. Deploy buttons work for simple Workers but fail with ~80% error rate for complex architectures. GitHub Actions provides reliable deployment with proper authentication and build handling.
